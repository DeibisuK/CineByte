Nuestro Proyecto

CineByte es un proyecto académico desarrollado por estudiantes de la Universidad Técnica de Machala que recrea digitalmente un sistema de gestión cinematográfica. Nuestra plataforma simula los servicios completos de un complejo de cines moderno, incluyendo:

  • Reserva de boletos en línea
  • Selección de asientos virtuales
  • Gestión de cartelera digital
  • Administración de salas simuladas

Nuestra Misión

Desarrollar una plataforma educativa de simulación cinematográfica que implemente soluciones tecnológicas en desarrollo web para recrear fielmente la experiencia de reservas y gestión de cines.

Como proyecto académico, nos comprometemos a ofrecer un entorno de aprendizaje práctico donde los estudiantes puedan aplicar sus conocimientos técnicos.

Tecnologías Utilizadas

El proyecto ha sido desarrollado utilizando tecnologías modernas de desarrollo web:

  • Frontend: Angular 18, TypeScript, HTML5, CSS3
  • Backend: Node.js, Express.js
  • Base de Datos: PostgreSQL
  • Herramientas: Git, Visual Studio Code

Objetivos Académicos

CineByte tiene como propósito demostrar las competencias adquiridas en:

  • Desarrollo de aplicaciones web full-stack
  • Diseño de interfaces de usuario modernas y responsivas
  • Implementación de bases de datos relacionales
  • Gestión de estados y navegación en aplicaciones SPA
  • Integración de APIs y servicios web

Nota Final

CineByte es una plataforma educativa sin vinculación comercial. Todos los cines, salas y servicios mencionados son parte de una simulación académica.
